Website Calibration

The calibration below is based on your reference comparison: website DLI = 2.35, website lux = 464, website PPFD = 31, actual DLI = 4.079592, actual lux = 712.3436354, and actual PPFD = 47.2175.

Recommended website logic: calibrated lux = raw lux x lux calibration factor; calibrated PPFD = calibrated lux x lux-to-PPFD factor; DLI = accumulated PPFD x elapsed seconds / 1,000,000.

Hybrid LED Override Control

Use this when the plant canopy becomes larger and the fiber light is not scattered enough. Manual Override and Full Brightness Mode allow the hybrid LED to operate independently of PPFD readings.
